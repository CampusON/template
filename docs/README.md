
## 패키지 설치 안내

### Node.js 백엔드
```bash
cd backend/nodejs-api
npm install
```

### Python 백엔드
```bash
cd backend/python-api
python -m venv venv
# (윈도우)
venv\Scripts\activate
# (맥/리눅스)
source venv/bin/activate
pip install -r requirements.txt  # requirements.txt가 있을 경우
```

### C# (.NET) 백엔드
```bash
cd backend/csharp-api
dotnet restore
```
